Assignment #1: BASICS

Friday, 30 June 2017

This assignment is simple. It consists of 5 tasks:

  1. Sign up for a GitHub account and log into CodeAnywhere (we did this on the first day of class).
  2. Take the first quiz.
  3. Take the beginning questionnaire.
  4. Return your signed grade contract.
  5. Read the syllabus all the way to the end. Look out for dinosaurs.

Once you have done these, you have completed the first assignment. You will likely have done these things by the time we finish class on Tuesday.
